# Configuração do Task Scheduler do Windows para atualização automática do ads.txt
#
# 1. Abra o Task Scheduler (taskschd.msc)
# 2. Clique em "Create Basic Task" no painel direito
# 3. Configure conforme abaixo:
#
# Nome: "Atualizar ads.txt Ezoic"
# Descrição: "Atualiza automaticamente o arquivo ads.txt da Ezoic"
#
# Trigger (Gatilho):
# - Daily (Diário)
# - Start: 02:00:00
# - Recur every: 1 days
#
# Action (Ação):
# - Start a program
# - Program/script: powershell.exe
# - Add arguments: -ExecutionPolicy Bypass -File "C:\caminho\para\seu\projeto\update-ads-txt.ps1"
# - Start in: C:\caminho\para\seu\projeto
#
# Conditions (Condições):
# - Desmarque "Start the task only if the computer is on AC power"
# - Marque "Wake the computer to run this task" (opcional)
#
# Settings (Configurações):
# - Marque "Allow task to be run on demand"
# - Marque "Run task as soon as possible after a scheduled start is missed"
# - Marque "If the running task does not end when requested, force it to stop"
#
# Para testar:
# 1. Clique com botão direito na tarefa criada
# 2. Selecione "Run"
# 3. Verifique o arquivo ads-update.log para confirmar execução
