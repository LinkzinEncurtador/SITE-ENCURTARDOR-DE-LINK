# Instruções para configurar o Windows Task Scheduler
# para atualizar a biblioteca anti-adblock do Adcash

## Passo 1: Abrir o Task Scheduler
1. Pressione Win + R
2. Digite "taskschd.msc" e pressione Enter

## Passo 2: Criar uma nova tarefa
1. Clique em "Create Basic Task..." no painel direito
2. Nome: "Atualizar Biblioteca Anti-Adblock Adcash"
3. Descrição: "Atualiza a biblioteca anti-adblock do Adcash a cada 5 minutos"

## Passo 3: Configurar o trigger
1. Selecione "Daily"
2. Clique em "Next"
3. Configure para começar hoje
4. Clique em "Next"

## Passo 4: Configurar a ação
1. Selecione "Start a program"
2. Clique em "Next"
3. Program/script: powershell.exe
4. Add arguments: -ExecutionPolicy Bypass -File "C:\caminho\para\seu\projeto\update-adblock-cron.ps1"
5. Start in: C:\caminho\para\seu\projeto

## Passo 5: Configurar repetição
1. Marque "Open the Properties dialog for this task when I click Finish"
2. Clique em "Finish"
3. Na aba "Triggers", clique em "Edit"
4. Marque "Repeat task every: 5 minutes"
5. For duration: "Indefinitely"
6. Clique em "OK"

## Passo 6: Configurar condições
1. Na aba "Conditions", desmarque "Start the task only if the computer is on AC power"
2. Marque "Start the task only if the following network connection is available: Any connection"

## Passo 7: Configurar configurações
1. Na aba "Settings", marque "Allow task to be run on demand"
2. Marque "Run task as soon as possible after a scheduled start is missed"
3. Marque "If the running task does not end when requested, force it to stop"

## Verificação
- Para testar: Clique com botão direito na tarefa e selecione "Run"
- Para ver logs: Abra o arquivo adblock-update.log no diretório do projeto
